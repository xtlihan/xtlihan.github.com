xilinx zedboard

创建PlanAhead工程

  1. 创建工程目录
    $ mkdir multi_uart
    $ cd multi_uart/
  2. 导入ise的环境变量
    $ source /opt/xilinx/14.6/settings64.sh
  3. 启动PlanAhead
    $ planAhead
  4. 创建名为multi_uart的工程，路径为/mnt/zedboard/ise/multi_uart
  5. 芯片选择xc7z020clg484-1
  6. 工程建好后，添加IP（Add or Create Embedded Sources）
    • 增加一个名为system的Sub-Design
    • 确定后弹出窗口，选YES即可
  7. 打开I/O Peripherse :
    • 设置Bank1 IO voltage为1.8V
    • 开启qspi flash
    • 开启eth0 use mio, and mdio
    • 开启usb0
    • 开启sd0, cd:mio 47, wp:mio46
    • 开启uart1
    • 开启gpio, mio50 mio51 in direction
  8. 打开DDR:
    • 选择MT41J128M16HA-15E
    • 开启Interal Vref
    • 开启DRAM Training: Write leveling, Read gate, Read data eye
    • 设置Pin Group Delay
